# 🏨 ダイナミックプライシング管理システム

## 那須ミッドシティホテル 価格管理・半自動承認システム

---

## 📌 概要

ホテルの客室料金をAI/ロジックが自動計算・提案し、最終判断は人間（ホテル担当者）が行う「半自動ダイナミックプライシング管理システム」のデモ版です。

---

## ✅ 実装済み機能

### 🏠 ダッシュボード

- KPI表示（稼働率・ADR・RevPAR・承認待ち件数）
- 需要シグナルアラートバナー
- 今後30日間の需要予測グラフ（Chart.js）
- チャネル別予約比率ドーナツグラフ
- 承認待ち提案のクイックテーブル

### 💡 価格提案ページ

- AI価格エンジンによる自動計算（曜日・稼働率・イベント・祝日・季節性）
- ステータスフィルター（全件／承認待ち／承認済み／却下済み）
- 部屋タイプフィルター
- 各提案カードに変更理由タグ・信頼度バー表示
- 一括承認ボタン
- 価格再計算ボタン

### ✅ 承認ワークフロー（モーダル）

- 変更根拠（理由リスト）の詳細表示
- 最終確認価格の手動調整入力（上下限バリデーション付き）
- 承認ボタン／却下ボタン
- 千円単位への自動丸め

### 📅 価格カレンダー

- 月単位の価格表示（承認済み・提案中・高需要・低需要を色分け）
- 日付クリックで詳細表示・その場で承認操作
- 前月／翌月ナビゲーション

### 📜 承認履歴

- 全承認・却下履歴の一覧表示
- キーワード検索
- CSVエクスポート機能

### ⚙️ 価格設定

- 部屋タイプ別 基準価格・**下限価格・上限価格**の設定
- 部屋タイプ別 自動承認トグル
- 曜日別係数設定（全7曜日）
- 稼働率連動係数設定（90%以上/70-89%/50-69%/50%未満）
- イベント・特日設定（追加・削除・係数調整）
- 変更幅制限設定（1回の最大値上げ額・値下げ額）
- 設定の永続保存（localStorage）

### 📂 CSVインポート

- CSV ドラッグ&ドロップ / ファイル選択
- プレビュー表示（先頭5行）
- サンプルCSVダウンロード
- 手動データ入力フォーム（日付・部屋タイプ・稼働率・ADR・チャネル）

## 🏗️ ファイル構成

```
index.html              メインHTML
css/
  style.css             全スタイル（レスポンシブ対応）
js/
  data.js               マスターデータ・設定・状態管理・ユーティリティ
  engine.js             価格計算エンジン（需要シミュレーション・係数計算）
  ui.js                 UI描画・承認ワークフロー・各ページレンダリング
  charts.js             Chart.jsグラフ描画
  app.js                アプリ起動・イベントバインディング
README.md               本ファイル
```

## 💾 データ保存

- `localStorage` に以下のキーで保存:
  - `dp_settings` : 設定データ
  - `dp_proposals` : 価格提案データ
  - `dp_history` : 承認履歴
  - `dp_manual_data` : 手動入力データ
